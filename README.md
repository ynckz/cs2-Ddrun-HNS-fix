# cs2-Ddrun-HNS

simple ddrun plugin on CS# for HNS Servers

Features:
- No stuck playback
- Support for the default +duck binding
- Only works with the mouse wheel
- Possibility to use SGS

bind for use:

	bind "MwheelDown" "+duck"

video demonstration - https://youtu.be/ImZqG9iHMyE
